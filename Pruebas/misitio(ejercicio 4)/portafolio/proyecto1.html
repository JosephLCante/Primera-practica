Descripción del Proyecto

La "Aplicación de Gestión de Tareas" es una herramienta diseñada para ayudar a los usuarios a organizar y gestionar sus tareas diarias de manera eficiente. Con una interfaz intuitiva, permite añadir, editar y eliminar tareas, así como marcarlas como completadas.

Características

  • Crear nuevas tareas
  • Editar tareas existentes
  • Eliminar tareas
  • Marcar tareas como completadas
  • Organizar tareas por categorías

Tecnologías Utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • Bootstrap
